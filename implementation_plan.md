# Внедрение AI-проверки ЕГЭ через Gemini 2.0 Flash

## Что меняется

Сейчас проверка делается через `mock-check.ts` (фиктивные баллы). Заменяем её на реальную проверку через Gemini 2.0 Flash для 4 предметов: **Русский язык, Обществознание, История, Английский язык**.

Пользователь получит:
- **Итоговый балл** + разбалловку по каждому критерию ЕГЭ
- **Подсветку мест** в тексте, на которые обратил внимание ИИ
- Комментарий по каждому критерию

---

## Открытые вопросы к вам

> [!IMPORTANT]
> **1. Задания в свободном поле или выпадающий список?**
> Сейчас пользователь вводит номер задания вручную (например, "27"). Лучший вариант — заменить на **выпадающий список** с конкретными типами заданий для каждого предмета (например, для Русского: "Сочинение К1–К12", для Обществознания: "Задание 25 — план", "Задание 26", "Задание 29 — эссе"). Это позволит автоматически применять правильный промпт и критерии.
> Оставить как есть (текстовое поле) или сделать список?

> [!IMPORTANT]
> **2. Подсветка в тексте — как показывать?**
> Для фичи "ИИ отметил это место" есть два варианта:
> - **Вариант A (проще):** Список цитат из ответа с пояснением под текстом
> - **Вариант B (красиво):** Выделение прямо в тексте ответа (жёлтая/красная подсветка)
> Какой вариант предпочесть?

> [!NOTE]
> **3. Статус сабмишена во время проверки**
> Запрос к Gemini занимает 3–8 секунд. Два подхода:
> - **Синхронно:** пользователь ждёт на странице, потом редирект (текущая логика)
> - **Асинхронно:** создаём сабмишен со статусом "pending", редирект сразу, страница результата показывает лоадер и polling
> Рекомендую **синхронно** — проще, а 5–8 секунд ожидания с красивой анимацией норма.

### 2. AI-модуль — промпты и логика

#### [NEW] app/lib/ai-check.ts
Основной модуль. Принимает `{ subject, taskType, answer }`, возвращает структурированный результат:
```typescript
{
  score: number,
  maxScore: number,
  strengths: string,
  improvements: string,
  mistakes: string,
  recommendation: string,
  criteriaScores: { code: string, name: string, score: number, max: number, comment: string }[],
  highlights: { text: string, note: string, type: "positive" | "negative" | "neutral" }[]
}
```

#### [NEW] app/lib/prompts/russian.ts
System prompt + критерии К1–К12 для сочинения по русскому языку (ЕГЭ 2024).

#### [NEW] app/lib/prompts/social-science.ts
Промпты для заданий 25 (план), 26, 27, 28, 29 обществознания.

#### [NEW] app/lib/prompts/history.ts
Промпты для заданий 18–19, 20, 21 истории.

#### [NEW] app/lib/prompts/english.ts
Промпты для заданий 37–40 (письмо, эссе) английского языка.
